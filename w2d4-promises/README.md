## Anatomy of a promise

Promise is an object!

Like an array, has some methods available

```jsx
  const arr = [1,2,3]

  arr.length

  arr.map()

  arr.forEach()
```

```jsx
  const promise = new Promise()

  promise.then()

  promise.catch()

  promise.finally()
```
